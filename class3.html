What is a library?

  • Software libraries hold functions (not books!)
  • When you include a library, you can use all the functions in that library
  • That means: you get to take advantage of other people's experience!
  • And... Save time!

How it works

  • Okay, I'll explain it!
  • We load jQuery first, then our plugin.
  • The plugin is two files: .js and .css
  • The JS handles the movements, click functions, and figuring out how to loop through.
  • The CSS handles the... css.

Let's develop it!

  • Scroll down a bit on the slider page.
  • Three things to do:
    • Include links to the .js and .css files (they are in the zip file)
    • Create the elements in the DOM in a list.
    • Activate the plugin in the jQuery section.
  • Refresh your page and ... Magic!
